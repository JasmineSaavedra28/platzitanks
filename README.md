# Platzi TANKS
Game for the Platzi GAME JAM 2022.

Based on the game TANKS! by Unity Learn, adapted to represent the Platziverse.

Platform: Windows

Players: 2

The game is in 3D, so the slight pixel art look was implemented with an in-camera shader.
